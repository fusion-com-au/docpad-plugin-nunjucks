# Nunjucks Plugin for [DocPad](http://docpad.org)

Projects your document's content with nunjucks adding the `nunjucks` extension to it

Convention:  `.*.nunjucks`



<!-- INSTALL/ -->

## Install

``` bash
docpad install nunjucks
```
